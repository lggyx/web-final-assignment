# 大学生通用AI技术分享网站

## 项目概述

这是一个面向大学生的AI技术分享静态网站，旨在提供一个简洁、易用的平台来分享人工智能相关知识和技术。

## 开发进度记录

- 创建说明文档
- 制定开发计划

- 添加CSS样式
- 添加JavaScript交互功能

## 技术栈

- HTML5
- CSS3
- JavaScript
- 响应式设计

## 网站功能规划

1. 首页展示
2. AI技术分类介绍
3. 学习资源推荐
4. 实践案例分享
5. 联系与反馈
